# TopUp API

A scalable RESTful API for a digital voucher marketplace built with **Express.js**, **TypeScript**, **MongoDB**, and **Midtrans Payment Gateway**.

> Secure authentication, voucher management, payment integration, and transaction reporting in one backend service.

# Features

## Authentication

- User Registration
- User Login
- JWT Authentication
- Password Hashing (bcrypt)

---

## Voucher Management

- Create Voucher
- Update Voucher
- Delete Voucher
- Get Voucher List
- Voucher Detail
- Search & Filter Voucher

---

## Transactions

- Create Transaction
- Transaction Detail
- Transaction History
- Order Tracking

---

## Midtrans Integration

- Snap Payment
- Payment Notification
- Automatic Payment Status Update
- Secure Server Validation

- Transaction Report

# Installation

```bash
git clone https://github.com/yourFEdev/backend-voucherHub.git
```

```bash
cd backend-voucherHub
```

Install dependencies

```bash
npm install
```

Run development server

```bash
npm run dev
```

Build project

```bash
npm run build
```
