# Backend API for Topup App with payment integration

RESTful API built with **Express.js**, **TypeScript**, **MongoDB**, and **JWT Authentication**.

## 🚀 Tech Stack

- Express.js
- TypeScript
- MongoDB
- Mongoose
- JSON Web Token (JWT)
- bcrypt
- dotenv
- cors


## 📌 API Modules

### Authentication

- Login
- Register
- JWT Authentication

### Voucher

- Create Voucher
- Get Voucher
- Update Voucher
- Delete Voucher

### Transaction

- Create Transaction
- Get Transaction History
- Transaction Detail

### Report

- Sales Report
- Transaction Report

### Midtrans

- Payment Integration
- Payment Notification
